# 🤖 AI Resume Ranker

An AI-powered Applicant Tracking System (ATS) built with Streamlit.

## Features

- 📄 Upload multiple resumes
- 🤖 Semantic AI matching
- 🏆 Candidate ranking
- 💻 Skill detection
- 📊 Dashboard analytics
- 📥 CSV report export

## Technologies

- Python
- Streamlit
- Sentence Transformers
- Scikit-learn
- PDFPlumber

## Run Locally

pip install -r requirements.txt

streamlit run app.py
